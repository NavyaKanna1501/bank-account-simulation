# Bank Account Simulation

A Python project demonstrating OOP concepts through a bank account management system.

## Features

- Base BankAccount class with encapsulation
- SavingsAccount with interest calculation
- CheckingAccount with overdraft protection
- Transaction history tracking
- Unit tests with unittest framework
